# Premium Animated Portfolio Website

A modern and responsive portfolio website developed using Next.js, TypeScript, Tailwind CSS, Framer Motion, and GSAP. This project was created to practice and demonstrate frontend development skills including UI design, animations, responsive layouts, and component-based architecture.

> Note: This is a sample portfolio project developed for learning and showcase purposes. The content, portfolio details, and personal information displayed are placeholders and do not represent my personal portfolio.

## 🚀 Technologies Used

* Next.js 15
* TypeScript
* Tailwind CSS
* Framer Motion
* GSAP
* Lucide React
* next-themes

## ✨ Features

### User Interface

* Modern and professional design
* Responsive layout for all screen sizes
* Dark and light mode support
* Smooth scrolling navigation
* Glassmorphism and gradient effects

### Website Sections

* Hero Section
* About Section
* Skills Section
* Projects Showcase
* Experience Timeline
* Services Section
* Testimonials Carousel
* Contact Form
* Footer

### Animations

* Page transitions
* Scroll-triggered animations
* Hover effects
* Text reveal effects
* Animated counters
* Stagger animations
* Floating elements

## 🛠️ Installation

### Clone the Repository

```bash
git clone <repository-url>
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
